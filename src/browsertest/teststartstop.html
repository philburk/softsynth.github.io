Report init(), start(),  stop(), and destroy() in Java Applet

When an Applet is launched, the browser calls the Applet's init() method() and its start() method. When the user leaves the page with the Applet,  the Applet's stop() method is called. At some point the Applet's destroy() method may be called when the browser wants to clean up.

This Applet write "init", "start", "stop" and "destroy" to a TextArea, and to the Java Console.

Note that Netscape calls the Applet's stop() then start() method whenever the user resizes the window.
