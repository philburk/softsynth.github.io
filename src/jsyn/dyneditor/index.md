## Using the Editor

To add a 12-tone equal temper note:

1.  Select the "Add" mode in the toolbar or hit 'A' key.
2.  Click in a clear area.

To add a just intoned note that is relative to another note:

1.  Select the "Add" mode in the toolbar or hit 'A' key.
2.  Click on an existing note and drag to where you want the new note.

To change the link between notes:

1.  Select the "Relink" mode in the toolbar or hit 'R' key.
2.  Click on the note you want to change and drag the rubber band to the note that you to link to.

To select a group of notes:

1.  Select the "Select" mode in the toolbar or hit 'S' key.
2.  Drag a rectangle around the notes you want.

To move a group of notes:

1.  Drag select one or more notes.
2.  Click and drag one of the notes.

To hear all of the song:

1.  Click before the "0" beat in the "beat bar" so there is no selected region.
2.  Click "Start".

To hear part of the song:

1.  Drag select a region in the "beat bar" above the note area.
2.  Click "Start".
