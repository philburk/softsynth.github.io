### Beta Release Notes for Dynamic Just Intonation Editor

### V0.0.9 - Beta 16 - 11/7/11

*   FIXED: drag selecting note was broken on Windows if you were scrolled to the right. Now fixed.
*   When you move the mouse over a note the popup window now shows the ratio and prime factors of the ratio relative to the very first note.

### V0.0.8 - Beta 15 - 11/4/11

*   Added 8/7 for Larry.
*   Added 8/5 for a "minor 6th".

### V0.0.7 - Beta 14 - 11/3/11

*   Added key commands 'S', 'A', and 'R' to quickly change modes.
*   Now BACKSPACE and DELETE key will delete selected notes.
*   Last note added is left selected.
*   Removed Delete mode.
*   Protect against saving a WAV file without a ".wav" suffix.

### V0.0.6 - Beta 12

*   Added "Record" button.
