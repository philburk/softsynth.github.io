What is JSyn?

JSyn stands for Java Synthesizer. It is an audio synthesis toolbox for Java programmers.

Java is a programming language from SUN that runs on almost all machines, including Macintosh, PC, and Linux. Java also can run in a web browser.

JSyn provides "unit generators" that create and process sound. These include oscillators, filters, envelopes, and sample players.

Units can be connected together to make complex sounds. The sound can be controlled very precisely.

JSyn provides a plugin for Internet Explorer and Firefox web browsers on Macintosh and PC. This allows you to play JSyn compositions at home.

JSyn also has a development kit for programmers with online tutorials.

JSyn can be downloaded freely from: http://www.softsynth.com/jsyn/

Examples: Plucked String, WaveMaker, Birds, Wind
