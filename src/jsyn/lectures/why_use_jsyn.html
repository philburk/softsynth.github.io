Why do Composers use JSyn?

JSyn is a flexible tool that does not impose any particular style.

A JSyn composer is also a Java programmer.

Java is a modern, powerful language that allows the composer to express complex ideas.

Java has other tools, for example 3D graphics, networking, and speech recognition, that can be combined with JSyn.

JSyn compositions can be placed in a web page that reaches a global audience.

The JSyn audience can be allowed to perform the composition themselves.

JSyn works on Macintosh, Windows, and Linux.

JSyn is being taught at many colleges and universities including Dartmouth, Cal Arts, Oberlin, New York University. Students learn Java which is a very useful skill.
