This Applet can be used to diagnose problems with installing the JSyn plugins. You should see all SUCCESS messages in the text box below. If you see ERROR messages or an empty box , then please look for an explanation below. For more information, see the support page.

Explanation of Error Messages

ERROR #1 or an empty box - Could not load JSyn Classes!

The JSyn classes could not be found by the browser or Java Virtual Machine.
If you have not already installed the JSyn plugin then please visit the plugins page.
If you have already installed the JSyn plugin then please try exiting completely from your browser and then restarting the browser.
If you are using the Sun Java Plugin on Windows then click here.

ERROR #2 or an empty box - Could not load com.softsynth.support.JSynDiagnostic class!

ERROR #3 - Could not load JSyn DLL or Library!

The JSyn DLLs or Extension could not be found by the browser or Java Virtual Machine. This could mean that the "npjsyn.dll", or "JSynV??.DLL" or the "JSynNative" file is is missing or in the wrong directory.
On a PC, it might mean that JDirect or JNI should be used in place of the current native interface.
If you are using the Sun Java Plugin on Windows then click here.

ERROR #4 - Could not start JSyn in NON-real-time mode!

If you got this far then all of the files are probably in the right place. But a failure here could be because another Applet already opened JSyn, but has not closed it properly. If you see a message about "openCount", then try quitting your browser and running this Diagnostic Applet again. If that works then please report the problem of "not calling Synth.stopEngine()" to the author of the Applet.

ERROR #5 - Could not start JSyn for output in real-time mode!

If this error occurs then there is probably some problem with the audio device. Try restarting your computer in case your audio device is hung.

WARNING #6 - Could not start JSyn for input!

Your audio hardware may not support audio input. Note that the old JSyn V14.1 does not support audio input on the Macintosh. Please upgrade to the latest version.

WARNING #7 - Could not start JSyn for simultaneous input AND output!

Your audio hardware may not support full duplex audio. Non-PCI sound cards on the PC, including Creative Labs AWE-64 and earlier, do not support full duplex audio.

If you need to report problems with installing JSyn then please cut and paste the error report from this Applet into your e-mail.
