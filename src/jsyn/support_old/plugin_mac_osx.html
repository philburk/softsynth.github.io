For  JSyn Plugins on Macintosh OS X

This plugin works well with Safari and Internet Explorer. There have been mixed results with other browsers.

When you install the plugin it should install two files into your {Home}/Library/Java/Extensions folder. They are:

  • JSynClasses.jar
  • libJSynV142.jnilib

If you are having trouble running JSyn Applets from a Mac OS X browser, please look in that folder by:

  1. Opening Finder
  2. Clicking Home icon
  3. Opening "Library" folder
  4. Opening "Java" folder
  5. Opening "Extensions" folder

Please let us know whether those JSyn files are in the Extensions folder. If not then you may wish to try reinstalling the JSyn plugin.
