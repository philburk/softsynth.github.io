For JSyn SDK on Windows (PCs)

Troubleshooting

If you get the runtime message "could not load native library for JSyn V###", where ### is the version number, then you might be missing the JSyn DLL. Look in your SYSTEM folder for a file called "JSynV###.DLL". On Windows 95 and Windows 98, the SYSTEM folder is "C:\WINDOWS\SYSTEM". On Windows NT is is called "C:\WINNT\SYSTEM32". If the DLL is missing, please copy it from your JSyn SDK "lib" folder to your SYSTEM folder.

If you cannot find an answer to your question on this website, then please contact us directly.
