JSyn and JMSL
Java Tools for Music Synthesis and Composition
by Phil Burk and Nick Didkovsky of SoftSynth.com
  • JSyn is a real-time audio synthesis API for Java.
  • Support for Mac and PC, plugins for IE and Netscape.
  • Used for teaching, experimental music, telecom apps, etc.
  • Wire is a graphical patch editor for JSyn.
  • Wire can export audio circuits as Java source code.
  • Available at:

    •     http://www.softsynth.com/
