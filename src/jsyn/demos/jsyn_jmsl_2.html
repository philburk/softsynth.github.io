JSyn and JMSL
Java Tools for Music Synthesis and Composition
by Phil Burk and Nick Didkovsky of SoftSynth.com
  • JMSL, the Java Music Specification Language, 

    • is a high level music composition toolbox that uses JSyn.
  • JMSL supports hierarchical scheduling and algorithmic composition.
  • JScore is an extensible score editor written using JMSL and JSyn.
  • Available at:
  •     http://www.algomusic.com/
