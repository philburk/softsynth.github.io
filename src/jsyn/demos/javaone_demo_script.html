Demo

  1. Hello. I'd like to demonstrate a Java API for Audio Synthesis called JSyn and a Music Composition API called JMSL.
  2. JSyn lets you create new sounds by connecting together oscillators, filters, sample players or other sound modules.
  3. You can also record and playback audio.
  4. JSyn can be used for developing music applications, sonifying web pages or for developing web based telecommunication apps.
  5. You can download the SDK for free from "www.softsynth.com". SoftSynth stands for "software-synthesizer".
  6. An example JSyn program is the WebDrum - collaboration server - jam together over the web.
  7. When one person changes a drum pattern, it shows up on all the other players screens. People can also chat while they play.
  8. [Show WebDrum]
  9. New Sounds can be designed using Wire.
  10. [Show Wire]
  11. This sound is built using two oscillators, a filter and two envelopes. Once I get the sounds the way I want it, I can export Java source code for use in JSyn applications.
  12. [Play AnalogVoiceB, Tweak filter setting, Show envelope editor.]
  13. Show JMSL Demo Page
  14. The second API is JMSL, the Java Music Specification Language.
  15. It is a high level music composition toolbox that normally uses JSyn but can also use JMIDI or JavaSound.
  16. JMSL supports hierarchical scheduling of musical events, and algorithmic composition.
  17. JScore is an extensible score editor written by New York composer and programmer Nick Didkovsky using JSML.
  18. By using Java interfaces, Nick is able to import user classes that can transform and generate musical material within the notation editor.
