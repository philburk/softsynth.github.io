XY Controller using the JSynPlayer

This is a demonstration of the JSynPlayer. The JSynPlayer is a JSyn Applet that can be controlled by JavaScript. It allows a web designer to integrate JSyn sound control with HTML. It does not require any Java programming.

Please note that this XY page will only work using Internet Explorer on Windows.

Simple Web Theremin

We call this a "theremin" in honor of Leon Theremin who invented a pioneering instrument that gave continuous control over pitch and amplitude.

Move the mouse over the blue area to control the sound.
