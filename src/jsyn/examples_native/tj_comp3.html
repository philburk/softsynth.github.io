FM Example

How it works: ORIGINAL version.
In this simple example new  pitches are a random whole number ratio of the previous pitch.  The modulator frequency of the FM operator is set to the previous notes frequency.  There is no absolute fundamental frequency or pitch scale.
Use the Capture button on the scope to grab some signals from one of the voices.  Keep trying if you get no signal the first time.  Notice how the envelope signal determines the amplitude of the carrier.  Also notice how the modulator distorts the carrier waveform.
