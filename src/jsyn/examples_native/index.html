Check out the many exciting JSyn Applets in the User Gallery. If you have a JSyn based Applet in a web page that we can link to, please let us know!

Simple Example Applets

  • Wind - white noise through a low pass filter with random modulation.
  • SawFader - modulate a sawtooth wave.
  • WaveMaker - synthesize a waveform from sine waves.
  • SeeOsc - see all the various waveforms available. Hear the difference between band-limited and non-band-limited oscillators.
  • FMLab - edit and play 4 operator FM patches.
  • Plucked String - Karplus-Strong physical model of plucked string.
  • Birds - generate synthetic bird sounds.
  • ChebyshevSong - play an algorithmically generated song using a Chebyshev WaveShaper.
  • Rain - rain-like sound generated using granular synthesis.
  • FilterFun - hear the effect of various filters on different sounds.
