Install JSyn plugin for the Sun JRE

Several dialogs will soon appear asking you for permission to download the JSyn plugin.
Please read each dialog carefully and click "Yes" or "OK" when asked.


After installation is complete, you can try some example JSyn Applets.

If you got an error message saying "Optional package installation is aborted." and you see "java.version = 1.5.0_06" in the box above then you have been bitten by a Sun Java bug. To install the JSyn plugin by hand, please click here.

If you get a message saying "ERROR - the JSyn classes are MISSING!" then click here.

Or if you get a message saying "ERROR - the JSyn library is MISSING!" then click here.

Or if the box above is blank with a little red X in the corner then you need to install Java so please go here.
