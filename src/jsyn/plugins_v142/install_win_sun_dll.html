Install JSyn plugin by hand for the Sun JRE

If you tried to install JSyn for Sun Java and got the message "ERROR - the JSyn library is MISSING!" then it is because the automatic Java installer did not work. You can install JSyn by hand, which involves placing one file on your hard drive.

  1. Note the Java version number in the box above. It will be something like "1.6.0_01".
  2. Find the corresponding folder on your hard drive for the Java run time engine.
    It will be something like "C:\Program Files\Java\jre1.6.0_01".
  3. Download this "JSynV142.dll" file and save it in the "bin" folder, for example: "C:\Program Files\Java\jre1.6.0_01\bin\". To download the DLL right-click on the link and choose "Save Target As..." or "Save Link As..." from the popup menu. It may ask you to replace an existing file. Please do so.
  4. If you cannot save the DLL then it is because the old one is still in use. In that case save the DLL to your desktop, exit the browser then move the DLL to the "C:\Program Files\Java\jre_whatever\bin\" folder.
  5. Close all windows from this web browser then restart the browser.

After installation is complete, you can try some example JSyn Applets.

Please let us know if this does not work.
