Download JSyn for Internet Explorer for Windows

Step #1 - Download and Install

Now downloading the JSyn package as a CAB file, which is about 220 KBytes. It will expand to about 550KB after unpacking. Please wait for downloading to complete. On a voice modem this may take a minute or two.

  • You will eventually see a Certificate appear, please "Grant" permission to install the JSyn plugin.
  • Then follow the instructions given by the NanoInstaller.
  • If IE refuses to download the CAB archive, then you can try downloading the Zip archive and using it to install the JSyn plugin. Double click on the file and let the WinZip Wizard guide you through the installation. Or uncompress the file and double-click on "SETUP.EXE".
If, some day, you ever need to UNinstall the the current version of JSyn from Internet Explorer, go to the Downloaded Package Manager, then right-click on the "JSyn Audio" package and select "Remove". To visit the Package Manager, click here for Windows 95 or 98 or click here for Windows NT.

Step #2- Test

  • Proceed to the JSyn Plugin Quick Check page.
If you have any problems with installing the Plugin or running the example programs, please visit our technical support page.
