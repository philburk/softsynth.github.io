The free JSyn plugin adds a music synthesizer to your browser.

If you do not see an "Install JSyn Plugin" button then you are probably missing Java itself. Java is used to run Applets, which are small programs that run in the web browser. For information on how to install or enable Java, please click here.

If the "Install JSyn" plugin button does nothing then select a link from the list below:

  • If the java.vendor is Microsoft then click here.
  • If the java vendor is Sun Microsystems and you are on Windows then click here.
  • If the java.vendor is Apple and you have Mac OS X then click here.
