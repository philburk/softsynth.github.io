Install JSyn plugin by hand for the Sun JRE

If you tried to install JSyn for Sun Java and got the message "ERROR - the JSyn classes are MISSING!" then it is because the automatic Java installer did not work. You can install JSyn by hand, which involves placing two files on your hard drive.

  1. Note the Java version number in the box above. It will be something like "1.5.0_06".
  2. Find the corresponding folder on your hard drive for the Java run time engine.
    It will be something like "C:\Program Files\Java\jre1.5.0_06".
  3. Download this "jsyn.jar" file by right clicking on that link and selecting "Save link as...". Save it in the Java "lib\ext" folder, for example: "C:\Program Files\Java\jre1.5.0_06\lib\ext\". Note that IE6 will try to rename the "jsyn.jar" file to "jsyn.zip". That's OK, either name will work But you should rename the file to "jsyn.jar" so you don't end up with both files some day.
  4. Download this "JSynV142.dll" file by right clicking and save it in the "bin" folder, for example: "C:\Program Files\Java\jre1.5.0_06\bin\".
  5. Close all windows from this web browser then restart the browser.

After installation is complete, you can try some example JSyn Applets.

Please let us know if this does not work.
