Download JSyn Plugin for Netscape 3 & 4 on Macintosh

Warning: The Macintosh Netscape 3 & 4 plugin for JSyn is unstable and may hang after running 5-10 JSyn Applets. The problem appears to be specific to Netscape 3 & 4 because JSyn runs fine under Netscape 6, Internet Explorer and iCab which all use MRJ. For the Macintosh, I recommend using the JSyn plugin for MRJ.

Step #1 - Download

  1. Download the JSyn Plugin for Macintosh.
  2. If Netscape doesn't convert it automatically, then decompress the file using StuffitExpander or other archive program.

Step #2 - Install

  1. Double click on the "InstallJSynPluginForNetscape" icon.
  2. After installation is complete, click on the "Refresh Plugins" button below OR restart Netscape Communicator.

Step #3 - Test

  • Proceed to the JSyn Plugin Quick Check page.
If you have any problems with installing the Plugin or running the example programs, please visit our technical support page.
