Install JSyn plugin on Macintosh OS X

A dialog will soon appear asking you for permission to install the JSyn plugin.
Please click on the "Grant this session" button.


After installation is complete, you will need to exit and restart your browser before running any JSyn Applets.

Please note that buttons and other GUI components are often bigger on Mac OS X than on other systems. So some Applets may not be able to fit in their windows.

Click here for help if the installation fails.
