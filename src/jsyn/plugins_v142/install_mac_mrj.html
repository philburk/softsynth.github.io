Download JSyn Plugin for Apple MRJ

Install the plugin for Internet Explorer, Netscape 6, or iCab on Macintosh.

Step #1 - Download

  • Download the JSyn Plugin for MRJ.
  • If Internet Explorer tells you that "The attempt to load the file failed.", then please try the following:

    • Click down on the link in step 1 above and HOLD DOWN THE MOUSE BUTTON.
    When you see a popup menu, select the "Download Link to Disk" item. The file should download and you can continue with the steps below.
(According to Bill Becker, this download problem may be due to a bad "File Helpers" setting for ".sit" files in the IE preferences.)

Step #2 - Install

  1. If your browser doesn't convert it automatically, then decompress the file using StuffitExpander or other archive program.
  2. Open the new jsyn folder and double click on the "InstallJSynForMRJ" icon.

Step #3 - Test

  • Proceed to the JSyn Plugin Quick Check page.
If you have any problems with installing the Plugin or running the example programs, please visit our technical support page.
