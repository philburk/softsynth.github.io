Install JSyn plugin on Linux

Important: the JSyn plugin needs to be installed in a root directory. If you run the browser in user mode then this installation will probably fail due to a "permission denied" error. So you may need to exit and restart your browser in root mode. If you are using Netscape 7.0, then you could log in as root and enter:

    /usr/local/netscape/netscape

Then return to the JSyn plugins page.

A dialog will soon appear asking you for permission to install the JSyn plugin.
Please click on the "Grant this session" button.


After installation is complete, you will need to exit and restart your browser before running any JSyn Applets. If you are running as root, you may wish to switch back to user mode to increase security.

Click here for help if the installation fails.
