JSyn Plugin Quick Check


This is a quick check to make sure that the JSyn Plugin is installed correctly. It may take a few seconds for the Applet to load. You may need to exit and restart your browser for the plugin to be activated. Look for SUCCESS or ERROR below.

SUCCESS: If the plugin is installed correctly, then please visit the JSyn Examples page.

ERROR: Try bookmarking this page, exiting your browser, then restarting your browser and coming back to this page. The JSyn plugin installation is supposed to work. If you encounter any problems, please contact us so that we can fix the problem. You can also visit the JSyn Technical Support page.

NOTHING: If the Applet does not appear, it may be because JavaScript or Java is disabled.
