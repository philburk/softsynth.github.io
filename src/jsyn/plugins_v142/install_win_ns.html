Download JSyn for Netscape Communicator on Windows

Step #1 - Download

  • Download the JSynPlugin for Windows,
  • If asked whether to "Open It", or "Save It to Disk", select "Open It". Otherwise just save it to disk.

Step #2 - Install

  • If the installer does not launch automatically, then save the file, double click on the ".zip" file to decompress it using WinZip or other zip program. Then double click on the SETUP.EXE file to install the plugin.
  • After installation is complete, click on the "Refresh Plugins" button below OR restart Netscape Communicator.

Step #3 - Test

  • Proceed to the JSyn Plugin Quick Check page.
If you have any problems with installing the Plugin or running the example programs, please visit our technical support page.
