In addition to the rights granted under the free license, purchasing a JSyn Developer License will also:

1.  **Provide access to a [password protected website](/restricted/) containing**:

*   downloadable archives of the documentation and tutorials for offline reading,
*   **a version of Wire that exports Java source code for a patch**,
*   source for the Java portion of JSyn (not unit generators, which are in 'C'),
*   a DirectSound version of the native DLL for Windows,
*   an ASIO version of the native DLL for Windows and Macintosh with improved multi-channel support.

3.  Grant you rights to install JSyn on a small-scale commercial web site with less than US$20,000.00 gross revenue per year

Note that a JSyn Developer License does **not** include the right to redistribute JSyn components for commercial, or non-commercial, purposes. This includes distributing the JSyn JAR or libraries on a CD or in a downloadable archive. Contact [sales](/jsyn/developers/sales/) for information on a Redistribution License.

Credit Card and billing information will be gathered on a secure Verisign site.
