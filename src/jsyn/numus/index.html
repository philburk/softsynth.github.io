JSyn Gallery for NUMUS 2000
JSyn is an audio synthesis toolbox for Java. It allows Java programmers to create interactive compositions and instruments that can run in a web browser. JSyn was developed by Phil Burk of SoftSynth.com.

WebDrum at Dartmouth
WebDrum on Local Server		 Phil Burk Multiple performers can play live music together over the web. Performers can edit drum patterns, melodies, or complex sounds using graphical envelope editors.
