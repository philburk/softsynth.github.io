WebDrum

by Phil Burk (c) 1997 - 2000

This Java Applet allows several performers from around the web to jam together. The sounds are produced using JSyn - a Java Audio Synthesis API. The communication between Applets is being handled by custom server software.

Instructions:

  1. Warning: Netscape will restart the Applet if you resize this window. This will log you out. So please resize your window if necessary before logging in.
  2. Please wait for the Applet to load. It is about 190K so it may take a minute to load if you are using a telephone modem. When you see the Login button below, then it is ready for use.
  3. Login using your real name or a handle.(Scroll down to see Applet.)
  4. Choose to join an existing session, or create a new one. Please join an existing session if available.

Once inside a Drum session:

  • Add drums from drum choice box. There are three types of drums: 1) regular drums with a simple pattern editor. 2) Melodic drums with a pitch-time grid, and 3) envelope drums which let you control a synthesizer.
  • Click [OWN] to gain control over a drum that someone else owns.
  • To turn on or off drum beats, click on the single line drum pattern.
  • To turn on or off notes in a melody, click on a column in the melody grid. Only one note per column may be selected. To turn off a note, click on it.
  • To edit an envelope, clock on the boxes at the top to select the parameter, eg. frequency, or amplitude. Then click and drag points to move them. Click between points to add new points. Shift-click on a point to delete it. Time is represented by the horizontal axis. The parameter value is represented by the vertical axis. There is a maximum time value for the entire envelope.
  • If you cannot add more drums, delete one to make room.
  • Type in the text field at the bottom of the screen and hit enter to chat with other performers.

    • Experiment and have fun. If you cannot find someone to jam with, come back on the next hour, or if that fails, at noon Pacific Standard Time. Or e-mail phil and schedule a session.
    A "Login" button will appear below when the Applet is ready.
