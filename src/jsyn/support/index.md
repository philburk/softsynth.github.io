# JSyn Support

[Click Here for Help Running JSyn Applets in a Browser](/jsyn/support/browser_users/)

If you want to **report a bug** for JSyn then please use the [Issue Tracker on GitHub.](https://github.com/philburk/jsyn/issues)

## JSyn Mail List

Visit the [JSyn Mail List on Google Groups](https://groups.google.com/g/jsyn) to see the archives or to join. You can use your old email address or a gmail address on the list.

Please note that we were using a list at columbia.edu that is no longer well supported. So if you were on that list you will need to rejoin the new list at Google. Note that the old Columbia mail list archives from 2000 to 2020 have been lost. That is one reason we are switching to Google Groups.
