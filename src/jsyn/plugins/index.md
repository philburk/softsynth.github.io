## JSyn Plugin

**Please note that JSyn no longer needs a plugin. JSyn is now implemented using [Pure Java](/jsyn/beta). Developers can put the JSyn JAR file on their website. Then users will not need a plugin to run the JSyn Applets.**

If you feel you need the obsolete plugin for some reason then you can install it from [here](/jsyn/plugins/index_old/).
