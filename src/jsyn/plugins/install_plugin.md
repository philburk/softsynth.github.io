## JSyn Plugin

**Please note that the old native code plugin for JSyn is no longer supported. Inestead we have the pure Java JSy that does not require a plugin.**

**Developers can put the JSyn JAR file on their website.**

If you have installed the old JSyn plugin and wish to uninstall it then use this [uninstall tool](/jsyn/plugins/uninstall_plugin/).
