JSyn SineOscApplet


JSyn is a real-time software synthesis API written in Java by Phil Burk. JSyn outputs CD quality 16 bit stereo sound from synthesis circuits the programmer designs in Java. JSyn applications can run in a Web Browser with the plugin, like this one, and as stand-alone applications.
