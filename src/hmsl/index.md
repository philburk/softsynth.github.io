# HMSL - Hierarchical Music Specification Language

HMSL is a programming language for experimental music composition and performance. It was actively developed between 1986 to 1996. HMSL is an object oriented set of extensions to the Forth language for the Amiga and the Macintosh.

## [Download HMSL for Mac OS and explore the Guided Tours on GitHub](https://github.com/philburk/hmsl/blob/master/docs/README.md).

HMSL was created by [Phil Burk](/philburk/), [Larry Polansky](https://eamusic.dartmouth.edu/~larry/), and [David Rosenboom](https://davidrosenboom.com) at the Mills College [Center for Contemporary Music](https://performingarts.mills.edu/center-for-comtemporary-music/index.php). It was distributed by [Frog Peak Music](http://www.frogpeak.org/fpartists/fpburk.html).

The original distribution is out of print. But the [source code is available on GitHub](https://github.com/philburk/hmsl) and is being ported to modern day Macintosh and Windows.
