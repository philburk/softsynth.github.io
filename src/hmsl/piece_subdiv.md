# SUBDIV

<center>&lt;p&gt;&lt;/p&gt;&lt;hr width="100%"&gt;&lt;p&gt;&lt;/p&gt;</center>

A set of shapes are created, each having a different subdivision of a measure. Thus we have shapes with one note per measure, another with two notes per measure, and so on 3,4,5,6,7,8,9... A production then selects up to 8 of these at the beginning of each measure so that every measure has a different set of subdivisions.
