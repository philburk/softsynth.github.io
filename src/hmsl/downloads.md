If you have occasional crashes when compiling from the editor that do not occur when compiling from a file, then download this fix.

### HMSL Extensions for 56000 DSP - Missing, does anyone have this?

This unsupported 1991 project contains 56000 assembly code for real-time unit generator based synthesis.  It also has HMSL extensions to load and control the 56000 units.  It also contains a software manual and schematics for connecting an Amiga to a GMP card.  
**Requirements:**  Macintosh running HMSL plus Digidesign Sound Accelerator or **AudioMedia I** card.  (Sorry. **AudioMedia II or later not supported.**)  Also requires knowledge of 56000 programming fundamentals.  Warning: this is a difficult project to hack but several people asked for it so I am posting it "as-is".
