JForth is a 32 bit subroutine-threaded Forth for the Amiga.

The JForth users have kindly contributed their code for others to use.  Delta Research and the authors do not warrant the fitness of this code for any purpose. It is intended for purely educational purposes.

*   Martin Randall's [**JForth.org**](http://www.jforth.org/) Page - has lots of links and examples.
*   [Crono and .FPS from Jorge Acereda.](/jforth/crono_ja/) - accurate time measurement.
*   [ParseCLI from Bob Dickow](/jforth/bd_parsecli.txt) - command Line Parsing
