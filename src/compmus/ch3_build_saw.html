Building a Sawtooth using Sine Waves

How it works:
A sawtooth wave can be constructed from sine waves. In this Applet we start with a sine wave and add more and more harmonics until we get to something like a sawtooth. Click the "Add Harmonic" button to start building the sawtooth wave.

If you add lots of harmonics, and then move the frequency fader to a high value then you may hear aliasing. This is because the higher harmonics are above the Nyquist Rate.
