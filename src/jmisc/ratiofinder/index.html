Ratio Finder

by Phil Burk of  SoftSynth.com.

This Java Applet will find the closest whole number ratio that approximates a floating point ratio. Suppose you want to find a close approximation to PI/2. Enter 3.14159265 in the "Floating Point Numerator". Enter 2.0 in the "Floating Point Denominator". Enter 512 in Maximum Integer Numerator. Then hit "Calculate". You will see a list of whole ratios that approximate PI/2. The error is calculated as follows: 

error_percent = (abs((int_numer/int_denom) - (fp_numer/fp_denom)) * 100.0) / (fp_numer/fp_denom)
